Tutorial Overview
-----------------

These tutorials are designed to take you from basic usage to advanced
customisation of the AHGD platform:

**For New Users**
~~~~~~~~~~~~~~~~~

* :doc:`basic_usage` - Get started with the dashboard interface
* :doc:`data_analysis` - Learn to perform health data analysis

**For Analysts**
~~~~~~~~~~~~~~~~

* :doc:`data_analysis` - Advanced analysis techniques
* :doc:`custom_visualizations` - Create custom charts and maps

**For Developers**
~~~~~~~~~~~~~~~~~~

* :doc:`api_integration` - Integrate with external systems
* :doc:`performance_optimization` - Optimise dashboard performance

Tutorial Requirements
---------------------

Before starting the tutorials, ensure you have:

* AHGD installed and running (see :doc:`../getting_started`)
* Sample data loaded (run ``python scripts/download_data.py``)
* Basic familiarity with Python (for developer tutorials)
* Understanding of health data concepts (helpful but not required)

Tutorial Data
-------------

The tutorials use real Australian health data from:

* Australian Institute of Health and Welfare (AIHW)
* Public Health Information Development Unit (PHIDU)
* Australian Bureau of Statistics (ABS)
* Socio-Economic Indexes for Areas (SEIFA)

This data is automatically downloaded and processed when you run the
setup scripts.
